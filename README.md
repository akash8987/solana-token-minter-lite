# Solana Token Minter Lite

This is a flat-structure, easy-to-deploy web utility for minting custom SPL tokens on Solana. It is designed for developers who want to understand the mechanics of token creation without digging through complex folder hierarchies.

## Features
* Connect to Solana Devnet or Mainnet.
* Create a new Mint Account.
* Initialize Token Metadata.
* Mint initial supply to your wallet.

## Tech Stack
* JavaScript / HTML / CSS
* @solana/web3.js
* @solana/spl-token

## How to use
1. Add your private key or connect via a browser wallet.
2. Define your token decimals and supply.
3. Click "Mint" and view your transaction on Solscan.
